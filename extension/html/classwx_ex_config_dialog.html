Detailed Description

Offers a dialog to set several items in the config.

If you only specify a wxCANCEL button, the dialog is readonly. You can also use the dialog modeless (then you can use wxAPPLY to store the items in the config). When pressing the apply button OnCommandConfigDialog is invoked from wxExFrame.

Detailed Description

template<class T>
class wxExItemTemplateDialog< T >

Offers a dialog template to set several items.

If you only specify a wxCANCEL button, the dialog is readonly. When pressing the:

  • wxAPPLY button
  • wxOK, wxCANCEL button for a modeless dialog
  • a ITEM_BUTTON
  • a ITEM_COMBOBOX_DIR the method wxExFrame::OnCommandItemDialog is invoked.
